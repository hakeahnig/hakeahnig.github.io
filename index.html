ANYhoist goes ROS

Bachelor's thesis project

ANYhoist is a attachable transport for the four-legged robot ANYmal. This project integrated the hardware in a ROS environment with a corresponding touchscreen UI

Additionally, a ROS to MQTT bridge was implemented, in order to be able to add existing, MQTT-based systems into ROS.

ROS MQTT Python Linux Docker bokeh

ADIR Row Change

This project was completed in collaboration with ANT ROBOTICS as a preparation for the bachelor's thesis.

ADIR is a robot platform for agricultural purposes. Our objective was to automate a plant row change in different situations, used for autonomous spraying applications. To make development easier, a simulation was set up with Gazebo. This coontains a self-created ADIR model and as well as as a dimensionally accurate greenhouse.

Also a prototype for a physical GUI, as well as a preset creation tool were created.

ROS Python 3D SLAM Robot Navigation QT

Ravioli I

A ArduSub controlled Submarine with five degrees of freedom.

Created for the "Land and Water robotics" lecture, our group decided to build a submarine. The robot uses ArduSub, running on a PixHawk with an external base station connected through ethernet.

Pixhawk ArduSub

Glotzer I

A ArduPilot controlled Monowing with autonomous stabilization and calibration.

Out of personal interest that came up during the "flying robotics" lecture, a classmate and I decided to build a monowing from scratch. We used a PixHawk controller provided by the school for the stabilization. Additionally, a FPV Setup was integrated for first-person flying over longer distances.

The main structure is made from styrofoam, which we cut to shape using a self built hot wire cutter, running along wooden profiles. The electronics compartment is 3D printed.

Pixhawk ArduPilot OnShape Wing design Foam Cutting

RO(S)Bertha

A first ROS project.

RO(S)Bertha is a tracked robot with an extendable grappling arm. The robot is built around a Raspberry Pi and a simple 2D Lidar. With a custom movement driver and existing ROS packages, RO(S)Bertha can do simple 2D Slam. A camera is attached behind the grappling mechanism, giving the robot computer vision capabilities.

ROS Linux Vim 2D SLAM OpenCV

Voron 0.2

Building a 3D Printer

As a summer project, I decided to build a Voron 0.2, a 3D printer based on an open source design.
